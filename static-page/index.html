Data Copilot

AI-Powered Insights for a Smarter Singapore

Addressing Complex Scenario Requirements

In high-complexity scenarios such as industrial and urban computing, existing general frameworks (e.g., Langchain, RAG, Text2SQL) struggle to meet the demands for accuracy, flexibility, and practicality. This project aims to propose a systematic method that combines LLM capabilities with professional domain knowledge to break through the limitations of current technologies in data integration, semantic understanding, and logical reasoning, achieving reliable application implementation.

Improving Data Query and Decision-Making Capabilities

In complex scenarios with multi-database redundancy, inconsistent field semantics, and varying field purposes depending on context, this project establishes a field meaning mapping table and unifies the structure and connection logic between databases. This ensures that the system can accurately generate data query logic even when facing semantic ambiguity, structural redundancy, and opaque rules.

Innovation

Agentic Flow Guided Problem Parsing and Interactive Clarification

Through multi-round interactions to eliminate ambiguity and generate pseudocode, it verifies whether the data can satisfy the intent. This method effectively combines natural language with structured data to understand user query intentions in complex backgrounds.

Database Structure Analysis and Field Semantic Alignment

Establishing a field meaning mapping table and unifying the structure and connection logic between databases, this project ensures that the system can accurately generate data query logic even when facing semantic ambiguity, structural redundancy, and opaque rules.

API and Function Specification Definition, Refined Code Generation

Solidifying professional knowledge into rule function code ensures precision and controllability. The LLM is mainly used for selecting intuitive and well-defined functions and flexibly matching parameters according to the problem.

Case and Code Template-Driven Based on Typical Scenarios

Establishing a case library and mapping templates provides reference logic and implementation for cross-database queries and analysis.
